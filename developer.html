Transpilers using a code template¶

There is also an interface for transpilers that use a code template as a basis for generated code. This is given by the TemplateTranspiler class. Examples that use this are the C_Transpiler or the OMP_Transpiler. As a subclass of Transpiler these transpilers must also implement the … methods.
