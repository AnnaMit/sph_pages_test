Discretizer module¶

class Discretizer.Discretizer[source]¶

Bases: object

This class provides functions to reshape sympy variables into their corresponding discrete form

create_timestep_dict()[source]¶
Parameters

variables (tuple of sympy.core.symbol.Symbol) – A tuple containg all variables, that appear in the ODE. The specific variables are assumed to be a sympy Function with dependency on x_i(t).

Returns

A dicitonary mapping every discrete instance of a variable onto the corresponding variablelookuptable-index and timestep offset. I.E: x_1_nplus1 -> (0 , 1), x_2_n -> (1 , 0)

var_with_n()[source]¶
Parameters

var (sympy.core.symbol.Symbol) – The variable, that should be discretized. The variable is assumed to be a sympy Function with dependency on x_i(t).

Returns

A discretized version of the var in dependence of discrete timestep n.
